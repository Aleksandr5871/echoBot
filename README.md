# 🤖 Telegram Echo Bot

Простой Telegram бот на Java, который повторяет ваши сообщения (эхо-бот). Создан для демонстрации работы с Telegram Bot API и Gradle.

## ✨ Возможности

- 🔄 Отвечает тем же текстом, который вы ему отправили
- 📝 Логирует все сообщения и события
- 🚀 Работает в режиме long polling
- ⚙️ Простая конфигурация через переменные окружения

## 🛠️ Технологии

- **Java** 17+
- **Gradle** - система сборки
- **TelegramBots** - библиотека для работы с Telegram API
- **SLF4J + Logback** - логирование

## 📋 Требования

- JDK 17 или выше
- Токен Telegram бота (получить у [@BotFather](https://t.me/BotFather))
